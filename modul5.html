Contoh JavaScript

Untuk menampilkan data dalam JavaScript di halaman web, Anda dapat menggunakan metode DOM (Document Object Model) untuk memanipulasi elemen HTML. Pertama, Anda perlu mendapatkan referensi ke elemen HTML yang ingin Anda gunakan untuk menampilkan data, biasanya dengan menggunakan metode seperti document.getElementById() atau document.querySelector(). Setelah Anda memiliki referensi elemen, Anda dapat mengubah kontennya dengan mengatur properti innerHTML atau textContent dari elemen tersebut. Misalnya, jika Anda memiliki elemen <div id="output"></div> di HTML Anda, Anda dapat menampilkan data dengan JavaScript menggunakan kode berikut: document.getElementById('output').innerHTML = 'Data yang ingin ditampilkan';. Dengan cara ini, Anda dapat secara dinamis memperbarui konten halaman web berdasarkan data yang Anda miliki di JavaScript.
