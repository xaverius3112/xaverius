3 Cara Menampilkan Data dalam JavaScript

  1. innerHTML: Mengubah konten HTML dari elemen yang sudah ada.
  2. createElement: Membuat elemen baru dan menambahkannya ke DOM.
  3. Template Literals: Membuat string HTML kompleks dengan backticks (`).

Pencarian buku di bawah menggunakan kombinasi dari metode createElement dan innerHTML.
